# Ayush Garg

16-year-old building drones, satellites, and the software that flies them. FPV pilot and solderer; I work across embedded, computer vision, and full-stack.

### What I build

Autonomy and ground software for robotics and aerospace — onboard flight logic, computer-vision pipelines, and the operator tools that command them — plus AI applications I ship end to end. I like problems with a real physical constraint: power, link budget, pointing, safety.

### Selected projects

- **[MuraltZ CubeSat — Flight Software](https://github.com/Ayush1298567/MIT-BWSI-Cubesat)** — Raspberry Pi flight software that gates image capture on live IMU stability and nadir lock, scores frames on-board, and downlinks the highest-priority images over a throttled link. Built at MIT Beaver Works.
- **[MuraltZ CubeSat — Ground Control Station](https://github.com/Ayush1298567/MIT-Cubesat-Ground-Control-Station)** — Python/Flask ground station: SIFT mosaicking, SSIM change detection across passes, and A\* hazard-aware route planning, driven from a live operator dashboard. Runs end to end against a mock satellite.
- **[SEEKR](https://github.com/Ayush1298567/SEEKR)** — Local-first ground-control station and simulator for a search-and-rescue drone swarm: event-sourced mission state, deterministic replay, and a sandboxed local-AI advisor that can't touch the controls.
- **[slidegen](https://github.com/Ayush1298567/slidegen)** — AI slide generator that runs on a Claude Code subscription: Claude plans the deck, Gemini renders the imagery, and it exports editable PowerPoint or PDF. Local-first, cents per deck.

More: [cell-design-engine](https://github.com/Ayush1298567/cell-design-engine) (battery cell design over a real PyBaMM simulation), [Prophet](https://github.com/Ayush1298567/Prophet) (defensive security tooling), [recap](https://github.com/Ayush1298567/recap) (local-first meeting recorder).
